OUTPUT REQUIREMENT:
Return a STRICT JSON array of objects.

CRITICAL LINGUISTIC RULES:
1. TARGET SCRIPT (ABSOLUTE): ALL output must be written in the DEVANAGARI SCRIPT (U+0900–U+097F) ONLY.

3. GENDER MARKING:
   - **For Nouns Only:** Every noun MUST have its grammatical gender denoted immediately after the word in parentheses.
   - Use (m) for masculine and (f) for feminine.

4. VERB FORMS:
   - Use the standard INFINITIVE form (ending in -ना / -nā).
   - Do NOT denote gender for verbs.

5. NO ENGLISH LEAKS:
   - Avoid English loanwords unless the native term is obsolete.
   - If an English word is absolutely necessary (e.g., 'Train'), write it in Devanagari: ट्रेन (f).

6. CONTEXT SENSITIVITY:
   - If 'english_word' has a parenthetical (e.g., "Right (direction)" vs "Right (correct)"), translate ONLY for that specific meaning.

## Key Visual Differences

### Day Cards (White Background)
- **Color**: `#FFFFFF`
- **Border**: `#E0E0E0`
- **Shadow**: Subtle elevation
- **Purpose**: Primary container

### Activity Cards (Gray Background)
- **Color**: `#F8F8F8`
- **Inside**: Day cards
- **Purpose**: Group language activities

### Today Highlight
- **Border**: Blue `#4A90E2`
- **Background**: Light blue `#F0F7FF`
- **Badge**: "Today" label

## Interaction States

### Weekly Goals
- **Empty Day**: Dashed border, "Tap to add" message
- **Day with Goals**: Shows language cards with activities

### Weekly Overview
- **Collapsed**: Shows summary (2/3, checkmark if complete)
- **Expanded**: Shows detailed activity breakdown
- **No Activities**: Grayed out, not interactive
- **With Activities**: Shows chevron, clickable

# Weekly Goals - User Guide

## What's New?

The Daily Goals feature has been upgraded to **Weekly Goals**! You can now plan your language learning schedule for the entire work week (Monday through Friday).

## How to Set Weekly Goals

### Step 1: Open Profile Tab
Navigate to the Profile tab at the bottom of the app.

### Step 2: Find Weekly Goals Section
Scroll down to the "Weekly Goals (Mon-Fri)" section and tap to expand it.

### Step 3: Add Activities to Days

You'll see:
- **Top**: A visual calendar showing Monday through Friday
- **Bottom**: Activity buttons (Reading, Listening, Writing, Speaking, Conversation)

To add an activity:
1. Find the activity row you want (e.g., "Reading")
2. Click the **+** button under the day you want to add it to
3. A colored badge will appear in that day's column

**Example**: Want to do 2 reading activities on Monday?
- Click the + button under Monday in the Reading row
- Click it again
- You'll see a badge with "2" on it

### Step 4: Review Your Week

Each day column shows:
- **Colored badges** for each activity scheduled
- **Language icon** showing which language (e.g., ‡≤ï for Kannada)
- **Activity icon** showing the type (book, headset, mic, etc.)
- **Number** showing how many times

**Tap any badge** to remove/reduce the count.

### Step 5: Save

Click the blue "Save Weekly Goals" button at the bottom.

## How It Works in the Dashboard

### Daily View
- Open the Dashboard tab
- You'll see **"Today's Goals"** with a badge saying "From Weekly Plan"
- Only activities scheduled for today will appear
- Track your progress as usual

### Empty Days
If you haven't set goals for today:
- You'll see a message: "No goals set for today"
- Reminder to set up weekly goals in Profile

### Weekends
By default, weekends (Saturday & Sunday) have no goals scheduled. Enjoy your rest!

## Tips for Success

1. **Start Small**: Don't overcommit. Start with 1-2 activities per day.

2. **Be Consistent**: Try to do activities at the same time each day.

3. **Mix Activities**: Vary between reading, listening, speaking, and writing for balanced learning.

4. **Adjust Weekly**: Review and update your weekly goals every Sunday evening.

5. **Multiple Languages**: Switch languages in Profile to set different schedules for each language you're learning.

## FAQ

**Q: Can I set goals for weekends?**
A: Currently, weekly goals are for weekdays only (Monday-Friday).

**Q: What if I miss a day?**
A: No worries! Your weekly plan stays the same. Just continue with tomorrow's goals.

**Q: Can I set different goals for different languages?**
A: Yes! Each language has its own independent weekly schedule. Switch languages in Profile to configure each one.

**Q: How do I remove an activity?**
A: Simply tap the colored badge for that activity, and the count will decrease by 1.

**Q: Can I add more than 5 activities to one day?**
A: Yes, but we recommend keeping it reasonable (3-5 activities per day) for consistency.

**Q: Will my old daily goals be lost?**
A: The old daily goals system is still accessible in the backend for backward compatibility, but the UI now uses the weekly system.
